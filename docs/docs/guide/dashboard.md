# Dashboard

The Dashboard provides an at-a-glance overview of your translation activity.

## Metrics Cards

- **Total Translations** — Number of translations completed
- **Characters Translated** — Total characters processed
- **Average Time** — Mean translation duration  
- **Top Language Pair** — Most frequently used language combination

## Recent Activity

The bottom section shows your most recent translations with type indicators (text or file), language pairs, and timestamps.

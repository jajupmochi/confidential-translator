# Text Translation

## How to Translate

1. Select **source language** (or leave on "Auto Detect")
2. Select **target language**
3. Type or paste text in the left pane
4. Click **Translate** — results stream in real-time

## Features

- **Real-time streaming** — See tokens appear as the LLM generates them
- **Translation report** — Model used, time taken, tokens/sec
- **Copy button** — One-click copy of the translated text
- **Abort** — Stop long translations mid-stream
- **Custom languages** — Type any language name; the system validates it via AI

## AI Disclaimer

A notice appears after each translation reminding you that the output is AI-generated and should be verified.
